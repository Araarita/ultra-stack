A) BACKEND: agents/proposal_system/

Crear los siguientes archivos:

1) __init__.py - Package init

2) models.py - Pydantic models:
   - Proposal (todos los campos)
   - ProposalStatus enum
   - Category enum
   - Impact enum
   - RiskLevel enum
   - ExecutionResult

3) proposal_generator.py - Clase ProposalGenerator:
   - Analiza sistema cada 30 min
   - Usa BlackBox AI Claude Opus 4.7
   - Genera 3-5 propuestas
   - Categorias: capability, improvement, maintenance, insight, security, experience
   - Respeta quiet hours 23:00-07:00 Mexico
   
   Cada propuesta tiene:
   - title (max 100 chars)
   - description (max 500 chars)
   - benefit
   - category, impact, effort, risk_level
   - action_type (install, modify, create, scan, report, configure, deploy)
   - action_code (codigo Python ejecutable)
   - estimated_time_seconds
   - dependencies
   - rollback_strategy
   - success_criteria

4) proposal_executor.py - Clase ProposalExecutor:
   - Ejecuta propuestas aprobadas
   - Sandbox con timeout subprocess
   - Backup/snapshot automatico
   - Valida success_criteria
   - Rollback automatico si falla
   - Logging detallado
   - Valida contra security blocklist

5) proposal_store.py - Clase ProposalStore:
   - CRUD en PostgreSQL con asyncpg
   - Tabla proposals con indices
   - Metodos: create, get_pending, get_by_id, update_status, add_execution_log, get_stats, delete_old
   - Retencion: auto-delete >30 dias

6) config.py - Variables de entorno:
   - PROPOSAL_GENERATION_INTERVAL_MINUTES (default 30)
   - PROPOSAL_MAX_PENDING (default 5)
   - PROPOSAL_MIN_IMPACT (default medium)
   - PROPOSAL_AUTO_APPROVE_SAFE (default false)
   - PROPOSAL_QUIET_HOURS_START/END
   - PROPOSAL_TIMEZONE

7) scheduler.py - Scheduler async:
   - Respeta quiet hours
   - No genera si >5 pending
   - Adaptativo segun tasa de aprobacion

B) DATABASE MIGRATION: migrations/001_proposal_system.sql

CREATE TABLE IF NOT EXISTS proposals (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    title VARCHAR(100) NOT NULL,
    description TEXT NOT NULL,
    benefit TEXT NOT NULL,
    category VARCHAR(20) NOT NULL,
    impact VARCHAR(10) NOT NULL,
    effort VARCHAR(10) NOT NULL,
    risk_level VARCHAR(10) NOT NULL,
    action_type VARCHAR(20) NOT NULL,
    action_code TEXT NOT NULL,
    estimated_time_seconds INTEGER DEFAULT 60,
    requires_restart BOOLEAN DEFAULT FALSE,
    requires_approval_password BOOLEAN DEFAULT FALSE,
    dependencies JSONB DEFAULT '[]',
    rollback_strategy TEXT,
    success_criteria TEXT,
    status VARCHAR(20) DEFAULT 'pending',
    created_at TIMESTAMPTZ DEFAULT NOW(),
    responded_at TIMESTAMPTZ,
    executed_at TIMESTAMPTZ,
    execution_log TEXT,
    user_feedback JSONB
);

CREATE INDEX idx_proposals_status ON proposals(status);
CREATE INDEX idx_proposals_category ON proposals(category);
CREATE INDEX idx_proposals_created ON proposals(created_at DESC);
CREATE INDEX idx_proposals_impact ON proposals(impact);
